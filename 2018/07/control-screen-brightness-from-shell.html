Control Screen Brightness from shell

change screen brightness from cmdline

On Tue 17 July 2018 in Tips
#Bash #oneliners

If you are using Xmonad or other different desktop envioments and using gnome-settings-daemon

  $ sudo /usr/lib/gnome-settings-daemon/gsd-backlight-helper --set-brightness (0-1000 range)
